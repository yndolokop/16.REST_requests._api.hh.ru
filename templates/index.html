Парсер вакансий по навыкам на hh.ru

Информация о парсере


Парсер осуществляет поиск вакансий с hh.ru и сортирует их по навыкам.

Поиск всегда осуществляется по ключевому слову и с учетом навыков прописаных в описании Вакансии. Навыки выводятся при каждом запросе. Ключевое слово вводится в основном поле. Это может быть профессия или вид деятельности.

В фильтрах есть дополнительные параметры поиска такие как регион и область деятельности, а также общий параметр 'везде', при котором поиск будет осуществляться по всему сайту.

В фильтрах также можно сузить поиск по специализации, выбрав из выпадающего списка нужную область деятельности. Например, во втором окне можем ввести один или несколько навыков, которые нас интересуют.

Парсер отсортирует все навыки, которые были найдены по ключевому слову во всех вакансиях по нисходящей от более востребованных к менее востребованным.

Также планируется ввести поля для вывода 'id' вакансии и ссылки на саму вакансию на сайте hh.ru.
